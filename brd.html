Business Requirements Document

Success Criteria

Success will be gauged by improvements in average time spent to run code tests and other KPIs such as Test run setup time improvement %, Test responsibility diagnosis improvement time %, Test fix improvement time %, Test run waiting time % improvement, Average time savings per week per head, and Average annual labor savings.

Metric Name Description Current Goal % Change
Test run setup time Time to prepare and initiate a test run 30 min 3 min 90% Reduction
Test responsibility diagnosis Time to determine cause of broken test and assign responsibility 15 min 3 min 80% Reduction
Test run waiting time Time to run an entire test suite 1 hour 20 min 67% Reduction
Test fix improvement time Time to fix a broken test 30 min - 1 hour 3-5 min 90% Reduction
